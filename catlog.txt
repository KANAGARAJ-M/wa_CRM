Below is an example API request for fetching products in your catalog. Try making this call in GraphAPI Explorer.


curl -i -G \
 'https://graph.facebook.com/v24.0/1497474621350981/products' \
 -d 'filter={"name":{"i_contains":"title"}}' \
 -d 'fields=retailer_id,id,name,category,errors' \
